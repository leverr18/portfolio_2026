// UNIVERSITY — MACHINE LEARNING · TEAM OF 5
Uber vs Lyft Fare Comparison & Analysis

Built the complete ML pipeline to predict Uber and Lyft ride prices from a real-world dataset in Boston — handling data cleaning, feature engineering, model selection, training, and deployment. The process involved real experimentation: starting with basic regression models that underperformed, pivoting to a Random Forest Classifier, then identifying that the framing was wrong and returning to regression. The final Random Forest Regressor achieved an R² of 0.962 across features including distance, weather, surge pricing, time of day, and ride type. Pipeline serialized with pickle for deployment.
The finished webapp lets you input two locations and trip details to get predicted prices across all ride types.

// UNIVERSITY — DATA ENGINEERING · TEAM OF 4
Crowdfunding ETL Pipeline

Built a full ETL pipeline from raw Excel files into a structured, queryable PostgreSQL database. Extracted and transformed crowdfunding campaign data by cleaning dates, assigning category and subcategory IDs, splitting contact names, and linking everything through foreign key relationships. Designed the entity-relationship diagram, wrote the database schema, and validated the final import with SQL queries.
Once the database was live, ran analysis to surface trends in campaign performance — revealing that theater dominated high-pledge campaigns at 34.3%, that campaign activity peaked mid-2021 before a sharp 2022 decline, and that creative categories consistently outperformed niche ones in backer volume.

// UNIVERSITY — FULL STACK DATA APP · TEAM OF 3
Meteorite Landings — Interactive Dashboard

Built the Flask backend and SQL data layer for a full stack interactive dashboard exploring 45,000+ meteorite landing records. Designed the SQLite database, wrote the query helpers, and built the API routes that fed data into a Plotly-powered frontend with multiple interactive views.
The app lets users filter by year range and explore: an interactive heatmap showing global strike density, a sunburst chart breaking down meteorite classes by time period, a bar graph of the top 15 most common classes, and a full filterable data table. Key finding: L6 was the most common class from 1950–2013, with 15,251 recorded landings in that window alone.
